# Amazon Product Analysis 📊

## 📌 Project Overview
This project analyzes an Amazon product dataset containing 1000 products.  
The goal is to perform data cleaning and exploratory data analysis (EDA) to understand product trends such as pricing and ratings.

---

## 📂 Dataset
- File: `amazon_1000_products.xlsx`
- Contains:
  - Product Name
  - Category & Subcategory
  - Price (USD)
  - Estimated Monthly Sales
  - Rating
  - Number of Reviews

---

## ⚙️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

---

## 🧹 Data Cleaning
- Checked for missing values
- Verified dataset structure using `.info()`
- Checked duplicate records
- Ensured dataset consistency

---

## 📊 Exploratory Data Analysis (EDA)

### ✔ Summary Statistics
- Used `df.describe()` to understand data distribution

### ✔ Ratings Distribution
- Visualized frequency of ratings using bar chart

### ✔ Price Distribution by Category
- Used Seaborn histogram to analyze pricing trends across categories

---

## 📈 Visualizations
- Ratings distribution (bar chart)
- Price distribution per category (stacked histogram)
